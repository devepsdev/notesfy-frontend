Proyecto CRUD de Notas

Esta aplicación es un sistema CRUD (Crear, Leer, Actualizar y Borrar) de notas desarrollado con un backend en Spring Boot utilizando JPA, MySQL, Lombok y Validation. El frontend está implementado en Angular con TypeScript y Angular Router para la navegación.

Tecnologías utilizadas

  • 🔹 Backend: Spring Boot, JPA, MySQL, Lombok, Validation
  • 🔹 Frontend: Angular, TypeScript, Angular Router
  • 🔹 Comunicación: API REST
